# YKDB Blog Themes

这里存放 YKDB Blog 前台模板。后台“模板市场”可配置仓库 `ahaqcxz/blogthemes` 并从这里安装模板。

每个模板目录需要包含 `theme.json`、`head.php`、`foot.php`、`index.php`、`read.php`、`error.php`、`assets/css/style.css`、`assets/img/nopic.svg`。

## 模板列表

- `cybergrid`：赛博矩阵，深色霓虹 HUD 风，控制台首屏、数据终端列表、扫描线和入场动画。
- `holoedge`：全息边界，冷白与深海黑混合的全息科技风，信息舱首页、光幕列表和柔和过渡动画。
